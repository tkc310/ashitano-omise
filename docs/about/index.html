アシタノオミセ東京とは

アシタノオミセ東京は、東京都内で開店前・開店したお店の情報をいち早くお届けするメディアです。
飲食店・セレクトショップなど業種を越えたお店をご紹介しています。

「行ってみよう」のキッカケになるようなお店が見つかるメディアを目指しています。

お店のオーナー様へ

アシタノオミセ東京への掲載は無料です。
SNSへの配信も定期的に行なっているため、検索エンジン・SNS・口コミでの集客効果が期待できます。

また、開店からまもなくHPなどの準備ができていないケースにおいても、基本情報を提供する手段としても有効に活用いただけます。
