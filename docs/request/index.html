掲載依頼

アシタノオミセ東京への掲載は無料です。
SNSへの配信も定期的に行なっているため、検索エンジン・SNS・口コミでの集客効果が期待できます。

また、開店からまもなくHPなどの準備ができていないケースにおいても、基本情報を提供する手段としても有効に活用いただけます。

掲載までの流れ

当メディアに掲載されるまでの流れをご説明いたします。

  1. 掲載条件を確認

    当メディアでは掲載条件を設けており、条件に合致しない・同意いただけない場合は掲載を見送らせていただく場合があります。
    下記の条件をご確認ください。

    • 1ヶ月以内に開店予定、もしくは開店から1年以内のお店
    • 東京都内で営業予定、もしくは営業しているお店
    • いただいた文章は当メディアのイメージにあった文体に調整させていただく場合があります
      (文章の意図が変わらない範囲での調整です。ただし過激な単語は調整する可能性が高いです)
    • 原則、掲載情報の加筆修正は受け付けておりません
      (掲載ページの削除は可能です)
    • 反社会的な団体、及び左記と関係を持つ団体でない

  2. 掲載に必要な情報を準備

    掲載する際の写真や文章の準備をお願いします。
    詳しくは後述の「掲載に必要な情報」をご確認ください。

  3. 掲載に必要な情報をメールで送信

    準備した写真や文章をメールにて送信をお願いします。
    詳しくは後述の「文章、画像の送信方法」をご確認ください。

  4. 写真の再撮影、文章の加筆依頼

    いただいた写真や文章が後述の「掲載に必要な情報」のフォーマットに沿っていない場合などに、ご連絡を差し上げる事がございます。

  5. 掲載完了のお知らせ

    掲載URLと共に掲載が完了した事をお知らせするメールをお送りいたします。

掲載に必要な情報

掲載依頼のメールに下記のフォーマットを貼り付けて、文章をご記入ください。

件名

【掲載依頼】 お店の名前(店舗名をご記入ください)

本文

- ご担当者様の情報 -
【ご担当者様の氏名】: 依頼を行われるご担当者様の氏名をご記入ください
【ご担当者様の連絡先】: ご担当者様のメールアドレスをご記入ください

- お店の情報 -
【お店の名前】: 店舗名をご記入ください
【リンク】: 店舗のHPやFacebook、予約サービスのリンクがある場合はURLをご記入ください(複数可)
【TEL】: 店舗の電話番号をご記入ください
【FAX】: 店舗のFAX番号がある場合はご記入ください
【MAIL】: 店舗のメールアドレスがある場合はご記入ください
【開店日】: 開店日をご記入ください(例 2019/01/01(火)
【営業日・営業時間】: 営業日と時間をご記入ください(例 平日10:00〜20:00 土日10:00〜15:00
【定休日】: 定休日をご記入ください(例 祝日、12/30〜1/3
【決済方法】: 現金や電子マネーなどの決済方法をご記入ください(複数可)
【平均的な予算】: サービスに掛かる平均的な予算のをご記入ください(例 1,000円〜3,500円
【住所】: 店舗の住所郵便番号からをご記入ください(例 〒150-0043 東京都渋谷区道玄坂２丁目１
【アクセス方法】: 店舗までの簡単な道案内をご記入ください(任意項目です)

- お店の概要 -
コンセプトなどの概要をご記入ください。

- お店のおすすめメニュー -
一押しのメニュー名とその説明をご記入ください。

- お店スタッフの方々について -
店舗を運営されているスタッフの皆さんについてご記入ください。
趣味や性格、生い立ちなどを紹介すると、閲覧ユーザーが親近感を持ってくれやすくなります。

文章、画像の送信方法

画像のフォーマットについて

画像は次のフォーマットで撮影、zip圧縮してメールに添付ください。
出来るだけ解像度が高く、加工していないものが望ましいです。
手戻りを防ぐため、それぞれの種類についてアングルや距離を変えて3枚ずつ撮影ください。
拡張子は全てjpgもしくはjpegになります。

  1. メインの写真

    縦撮りでお願いします。
    トップページで表示される画像になります。
    お店の外観が望ましいですが、難しい場合は見栄えする写真で問題ありません。

  2. 店内の写真

    横撮りでお願いします。

  3. おすすめメニューの写真

    横撮りでお願いします。
    一つのメニューに絞っていただく事が望ましいですが、商品の属性上難しい場合はこれに限りません。

  4. スタッフの皆さんの写真

    横撮りでお願いします。
    仕事をされている風景でも、休憩中のフランクな風景でも問題ありません。
    スタッフの皆さんの人柄、お仕事の姿勢が伝わる写真が望ましいです。
